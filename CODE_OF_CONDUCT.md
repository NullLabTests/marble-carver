# Code of Conduct

This project is a place for serious, transparent research.

We expect all participants to:

- Treat failures and retractions as valuable contributions (not embarrassments)
- Be kind when pointing out gaps in someone else's carving
- Credit the original carver when building on previous attempts
- Never delete history -- only add dated, visible corrections

Harassment, bad-faith "gotcha" auditing, or using this project to attack
researchers will result in immediate removal.

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/)
where it does not conflict with the above.

*Transparency is a form of respect.*
